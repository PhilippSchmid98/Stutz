# 03 UI/UX Guidelines

## Main Screens

1. **Dashboard**
   - Total Monthly Income
   - Total Monthly Expenses
   - Surplus/Deficit (with color indicator)
   - Quick links to Planning and Tracking

2. **Budget Planning**
   - Tree view of expenses (expand/collapse)
   - Add/Edit/Delete categories and items
   - Income management (add/edit/delete sources)
   - Visual hierarchy: indentation, bolding, icons

3. **Variable Expense Tracking**
   - List of variable expense items
   - Select item → Log transaction (amount, date, time)
   - History of tracked spendings (current month)

4. **Analysis**
   - Compare planned vs. actual for each variable item
   - Bar/line charts for visual comparison

## User Flow
- Onboarding: Set up income and main categories
- Dashboard → Plan Budget → Track Expenses → Analyze
- All navigation via bottom nav bar or side drawer

## Design
- Material 3
- Responsive layouts
- Clear distinction between fixed/variable items
- Use color and icons for clarity
